#2 SETUP DIRECTADMIN

####################################################################################################
- For Setup Directadmin, Please Run Command Below In SSH With Root User
+ Command: yum -y install nano wget perl; wget http://hack-like.me/directadmin/setup.sh; chmod 777 setup.sh; ./setup.sh
####################################################################################################

#3 TURNOFF FIREWALL AFTER RESTART OR START VPS/SERVER

####################################################################################################
- After Restart Or Start VPS/SERVER Could You Can't Login To Directadmin Because FireWall Auto TurnOn
- To Solve This Problem, Please Run Command Below In SSH With Root User
+ Command:
@ CentOS6: service iptables stop
@ CentOS7: systemctl stop firewalld
####################################################################################################

####################################################################################################
